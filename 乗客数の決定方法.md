# 乗客数の決定方法

トラベラーのルールにおいて、乗客の内訳は「全体の人数を決めてから各クラスに割り振る」のではなく、**「上級」「中級」「一般」「冷凍」の4つのクラスごとに独立して判定を行って決定します**。

具体的な手順は以下の通りです。

### 1. クラスごとにダイスを振る
上級、中級、一般、冷凍のそれぞれのクラスについて、個別に **2D（6面ダイス2個）** を振ります。

### 2. 修正値（DM）を加算する
振ったダイスの出目に、以下の修正値を加算します。
*   **共通の修正:** 〈ブローカー〉などのスキル効果値、出発地や目的地の人口・スターポートのクラスによる修正、距離による修正などを全てのクラスに加算します。
*   **クラス固有の修正:** さらに、クラスごとに以下の専用修正を加算します。
    *   **上級船客 (High):** **DM-4** （数が少なく見つかりにくいため）
    *   **冷凍船客 (Low):** **DM+1** （安価で利用者が多いため）
    *   ※中級と一般にはこの固有修正はありません。

### 3. 旅客発生数表に当てはめる
クラスごとに計算した「2D ＋ 修正値」の合計を「旅客発生数表」に当てはめ、発生する人数のダイス（例：3D、5Dなど）を決定します。

### 4. 最終的な人数を決定
最後に、決定されたダイスをクラスごとに振り、実際の乗客数を算出します。

---
**例：ある星系での判定**
共通の修正が「+2」だった場合、各クラスの判定は以下のようになります。
*   **上級船客:** 2D ＋ 2（共通） **－ 4（上級）** ＝ 「2D - 2」で表を参照。
*   **中級船客:** 2D ＋ 2（共通） ＝ 「2D + 2」で表を参照。
*   **一般船客:** 2D ＋ 2（共通） ＝ 「2D + 2」で表を参照。
*   **冷凍船客:** 2D ＋ 2（共通） **＋ 1（冷凍）** ＝ 「2D + 3」で表を参照。

※なお「労働船客」の専用ロールはなく、彼らは運賃を労働で相殺する中級客として扱われます。
